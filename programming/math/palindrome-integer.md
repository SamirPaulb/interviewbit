# Palindrome Integer

https://www.interviewbit.com/problems/palindrome-integer


## Solution

```cpp
int Solution::isPalindrome(int A) {

    if(A<0)
        return false;

    long int temp = 0, temp2 = A;

    while(A) {
        temp = temp*10 + A%10;
        A /=10;
    }

    return temp==temp2;
}
```
