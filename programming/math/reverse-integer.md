# Reverse Integer

https://www.interviewbit.com/problems/reverse-integer


Reverse digits of an integer.
Example1: x = 123, return 321
## Solution

```cpp

int Solution::reverse(int A) {
    long r=0;
    while (A) {
        r = r * 10 + A % 10;
        if (r>INT_MAX || r<INT_MIN)
            return 0;
        A /= 10;
    }
    return r;
}
```
